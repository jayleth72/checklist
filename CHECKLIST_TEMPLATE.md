# Checklist Template

This is a template checklist that you can use for your projects.

## Project Setup
- [ ] Clone the repository
- [ ] Install dependencies
- [ ] Configure environment variables
- [ ] Review [project guidelines](GUIDELINES.md)

## Development
- [ ] Create feature branch
- [ ] Write code
- [ ] Add tests
- [ ] Update documentation

## Review
- [ ] Self-review changes
- [ ] Run linter
- [ ] Run tests
- [ ] Create pull request

## Deployment
- [ ] Review deployment checklist
- [ ] Deploy to staging
- [ ] Verify in staging
- [ ] Deploy to production
